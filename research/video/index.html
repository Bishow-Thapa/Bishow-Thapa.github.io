    MongoDB

    Implicitly

    docker run -d -p 27017:27017 --name mongodb mongo

    "Run a container in detached mode (-d), map port 27017 on the host to port 27017 in the container (-p 27017:27017), name the container mongodb (--name mongodb), using the official mongo image."

    How to set admin username, password, and database name explicitly

    docker run -d -p 27017:27017 --name mongodb \
  -e MONGO_INITDB_ROOT_USERNAME=admin \
  -e MONGO_INITDB_ROOT_PASSWORD=yourpassword \
  -e MONGO_INITDB_DATABASE=yourdbname \
  mongo
    • MONGO_INITDB_ROOT_USERNAME sets the admin username.
    • MONGO_INITDB_ROOT_PASSWORD sets the admin password.
    • MONGO_INITDB_DATABASE sets the initial database name (note: this only creates the database if you insert data into it; it does not create it empty)

    To check inside the container:

    docker exec -it mongodb mongosh
    • mongodb is the name of your Docker container running the MongoDB server.
    • mongosh is the MongoDB Shell, an interactive JavaScript and Node.js-based command-line interface (CLI) used to connect to and interact with MongoDB databases.
    Command What it does
    docker exec -it mongodb mongosh Runs mongosh directly inside the container; opens MongoDB shell immediately
    docker exec -it mongodb Invalid; missing command to run inside container
    docker exec -it mongodb bash Opens an interactive bash shell inside container; then you can run mongosh manually

    Then you can run:

    show dbs

# Switch to the test database:
use test

# 3. Insert a document into a collection:
db.createCollection("mycollection")

# 4. Now run:
show collections

    This lets you create and verify databases and collections interactively.

    In short, your current MongoDB Docker container has no admin user or password and uses the default database test until you create others. To have admin credentials and a specific database, restart the container with the proper environment variables.

    Note

    The reason you don‚Äôt see the test database listed when you run show dbs is because MongoDB only shows databases that contain data. The default test database is created implicitly but only appears in the list after you insert some data into it.

    Summary:

    Database Name Visible in show dbs? When does it appear?
    admin Yes Always (system database)
    config Yes Always (system database)
    local Yes Always (system database)
    test No (initially) After inserting data into the test db

    Additional notes:

    If you want to create a specific database on container startup, you can use the environment variable MONGO_INITDB_DATABASE when running the container. However, this only creates the database if you insert data into it during initialization (for example, via scripts in /docker-entrypoint-initdb.d/).

    Typical .env file example:

    # For MongoDB container initialization
MONGO_INITDB_ROOT_USERNAME=admin
MONGO_INITDB_ROOT_PASSWORD=secretpassword
MONGO_INITDB_DATABASE=mydatabase

# For your app to connect to MongoDB
MONGO_URI=mongodb://admin:secretpassword@localhost:27017/mydatabase
    • The first three variables are used by the MongoDB container on startup.
    • The MONGO_URI is used by your application to connect to the database.
    Why define both?
    • Container setup and initialization require the root username, password, and initial database name.
    • Your application needs the connection URI to connect and operate on the database.
    • Keeping them in .env files allows you to manage credentials and configuration securely and flexibly.

    Summary

    Variable Purpose Where used
    MONGO_INITDB_DATABASE Initial database name for MongoDB container setup MongoDB Docker container
    MONGO_INITDB_ROOT_USERNAME Root username for MongoDB container MongoDB Docker container
    MONGO_INITDB_ROOT_PASSWORD Root password for MongoDB container MongoDB Docker container
    MONGO_URI Full connection string for your app to connect Your application (Node.js, etc.)

    In conclusion:

    You should define both the database name (for container initialization) and the connection URI (for your app) in your .env files to properly configure MongoDB in Docker and allow your application to connect securely and correctly.

    Summary Table

    Scenario Connection URI Hostname Explanation
    App running on host machine localhost Connects to MongoDB container port exposed on host
    App running inside Docker container mongodb Connects via Docker internal network DNS
    reportUpload.jsx
    import React, { useState, useCallback, useEffect } from "react";
import { useSelector } from "react-redux";
import {
  Divider,
  Flex,
  Drawer,
  Form,
  Input,
  DatePicker,
  Upload,
  Card,
  Progress,
  Popconfirm,
} from "antd";
import {
  PauseOutlined,
  PlayCircleOutlined,
  StopOutlined,
  UploadOutlined,
  ReloadOutlined,
  InboxOutlined,
} from "@ant-design/icons";
import { isEmpty } from "lodash";
import CustomButton from "@shared/components/Button/CustomButton";
import withDrawer from "@shared/hooks/withDrawer";
import useDrawerReducer from "@shared/hooks/useDrawerReducer";
import {
  useUploadTreasuryReportsMutation,
  useStartUploadTreasuryReportMutation,
  usePauseTreauryReportUploadMutation,
  useResumeTreasuryReportUploadMutation,
  useRestartTreasuryReportUploadMutation,
  useCancelTreasuryReportUploadMutation,
  useGetTreasuryReportStatusQuery,
} from "@features/reporting/services/glTrialBalanceTodApi";
import logger from "@shared/utils/logger";
import dayjs from "dayjs";

const { Search } = Input;

const ReportUpload = () => {
  const currentPermission = useSelector((state) => state.auth.permissions);
  const [isLoadingButton, setIsLoadingButton] = useState(false);
  const [buttonState, setButtonState] = useState({
    start: false,
    pause: false,
    resume: false,
    restart: false,
    cancel: false,
    error: "",
  });
  const [progress, setProgress] = useState(0);
  const [fileInfo, setFileInfo] = useState(null);

  const { drawerVisible, drawerMode, onAdd, onClose } = useDrawerReducer();
  const [importform] = Form.useForm();

  const [
    uploadTreasuryReports,
    { data: uploadReportData, isLoading: isLoadingUploadReport },
  ] = useUploadTreasuryReportsMutation();

  const [startUploadTreasuryReport, { isLoading: isLoadingStartUpload }] =
    useStartUploadTreasuryReportMutation();

  const [pauseTreauryReportUpload, { isLoading: isLoadingPauseUpload }] =
    usePauseTreauryReportUploadMutation();

  const [resumeTreasuryReportUpload, { isLoading: isLoadingResumeUpload }] =
    useResumeTreasuryReportUploadMutation();

  const [restartTreasuryReportUpload, { isLoading: isLoadingRestartUpload }] =
    useRestartTreasuryReportUploadMutation();

  const [cancelTreasuryReportUpload, { isLoading: isLoadingCancelUpload }] =
    useCancelTreasuryReportUploadMutation();

  console.log(buttonState);
  const {
    data: reportStatusData,
    isFetching: isFetchingStatus,
    isError: isErrorReportStatusData,
    error: errorReportStatusData,
  } = useGetTreasuryReportStatusQuery(
    { processId: uploadReportData?.processId },
    {
      skip: !uploadReportData?.processId || !buttonState.pause,
      pollingInterval: buttonState.pause ? 5000 : 0,
    }
  );

  const updateButtonState = (state) => {
    setButtonState({
      start: false,
      pause: false,
      resume: false,
      restart: false,
      cancel: false,
      error: state?.errorMessage,
      ...state,
    });
  };

  const handleImportSubmit = async (values) => {
    setIsLoadingButton(true);
    setProgress(0);
    setFileInfo(null);
    try {
      if (values.dragger && values.dragger.length > 0) {
        const file = values.dragger[0].originFileObj;
        const fileFormData = new FormData();
        fileFormData.append("file", file);

        const response = await uploadTreasuryReports({
          file: fileFormData,
          reportDate: dayjs(values.reportDate).format("MM-DD-YYYY"),
        }).unwrap();

        setFileInfo({
          name: file.name,
          size: `${(file.size / 1024).toFixed(2)} KB`,
        });

        updateButtonState({ start: true });
        importform.resetFields();
        onClose();
      }
    } catch (error) {
      logger.error({ msg: "Upload Error: ", error });
    } finally {
      setIsLoadingButton(false);
    }
  };

  const handleStart = async () => {
    try {
      await startUploadTreasuryReport({
        body: {
          taskType: uploadReportData?.message,
          processorId: uploadReportData?.processId,
          parameters: uploadReportData?.parameters,
        },
      }).unwrap();
      updateButtonState({ pause: true, cancel: true });
    } catch (error) {
      logger.error({ msg: "Start Upload Error: ", error });
    }
  };

  const handlePause = async () => {
    try {
      await pauseTreauryReportUpload({
        processId: uploadReportData?.processId,
      }).unwrap();
      updateButtonState({ resume: true, restart: true, cancel: true });
    } catch (error) {
      logger.error({ msg: "Pause Upload Error: ", error });
    }
  };

  const handleResume = async () => {
    try {
      await resumeTreasuryReportUpload({
        processId: uploadReportData?.processId,
      }).unwrap();
      updateButtonState({ pause: true, cancel: true });
    } catch (error) {
      logger.error({ msg: "Resume Upload Error: ", error });
    }
  };

  const handleRestart = async () => {
    try {
      await restartTreasuryReportUpload({
        processId: uploadReportData?.processId,
      }).unwrap();
      updateButtonState({ pause: true, cancel: true });
      setProgress(0);
    } catch (error) {
      logger.error({ msg: "Restart Upload Error: ", error });
    }
  };

  const handleCancel = async () => {
    try {
      await cancelTreasuryReportUpload({
        processId: uploadReportData?.processId,
      }).unwrap();
      updateButtonState({ start: true });
      setProgress(0);
      setFileInfo(null);
    } catch (error) {
      logger.error({ msg: "Cancel Upload Error: ", error });
    }
  };

  useEffect(() => {
    if (reportStatusData) {
      setProgress(reportStatusData.progressPercentage || 0);
      updateButtonState({
        pause: reportStatusData.canPause,
        resume: reportStatusData.canResume,
        restart: reportStatusData.canRestart,
        cancel: reportStatusData.canCancel,
      });
      if (
        reportStatusData.status === "CleanedUp" ||
        reportStatusData.errorMessage
      ) {
        updateButtonState({ start: true });
        if (reportStatusData.errorMessage) {
          logger.error({
            msg: `Upload failed: ${reportStatusData.errorMessage}`,
          });
        } else if (reportStatusData.progressPercentage === 100) {
          logger.info({ msg: "Upload completed successfully" });
        }
      }
    }
  }, [reportStatusData]);

  const onImportCloseClick = () => {
    importform.resetFields();
    onClose();
  };

  const normFile = (e) => e?.fileList;

  const importCommonFields = (
    <>
      <Form.Item
        label="Report Date"
        name="reportDate"
        rules={[{ required: true, message: "Please select a report date!" }]}
      >
        <DatePicker
          allowClear
          style={{ width: "100%" }}
          format="YYYY-MM-DD"
          placeholder="Select Report Date"
          disabledDate={(current) => current && current > dayjs().endOf("day")}
        />
      </Form.Item>
      <Form.Item
        label="Dragger"
        name="dragger"
        valuePropName="fileList"
        getValueFromEvent={normFile}
        rules={[{ required: true, message: "Please upload a file!" }]}
      >
        <Upload.Dragger
          accept=".xls, .xlsx"
          name="files"
          maxCount={1}
          beforeUpload={() => false}
        >
          <p className="ant-upload-drag-icon">
            <InboxOutlined />
          </p>
          <p className="ant-upload-text">
            Click or drag file to this area to upload
          </p>
          <p className="ant-upload-hint">Support for a single upload.</p>
        </Upload.Dragger>
      </Form.Item>
    </>
  );

  return (
    <>
      <Flex justify="space-between">
        <Flex>
          <DatePicker defaultValue={dayjs()} style={{ width: "100%" }} />
          <Search placeholder="Search" allowClear style={{ width: "100%" }} />
        </Flex>
        <Flex gap={30}>
          <CustomButton
            type="primary"
            title="Create New Report"
            onClick={() => {
              importform.resetFields();
              onAdd();
            }}
            icon={<UploadOutlined />}
          />
        </Flex>
      </Flex>
      <Divider />

      {fileInfo && (
        <Card style={{ margin: 20, padding: 20 }}>
          <div
            style={{
              display: "flex",
              alignItems: "center",
              justifyContent: "space-between",
            }}
          >
            <div style={{ flex: 1 }}>
              <p>File: {fileInfo.name}</p>
              <p>File name: {fileInfo.name}</p>
              <p>File size: {fileInfo.size}</p>
            </div>

            <div style={{ flex: 2, textAlign: "right" }}>
              {buttonState?.error && (
                <div style={{ color: "red", marginBottom: 8 }}>
                  {buttonState.error}
                </div>
              )}
              {buttonState.start && (
                <CustomButton
                  type="primary"
                  title="Start"
                  onClick={handleStart}
                  icon={<UploadOutlined />}
                  loading={isLoadingStartUpload}
                />
              )}
              {buttonState.pause && (
                <PauseOutlined
                  onClick={handlePause}
                  style={{ marginLeft: 10, cursor: "pointer" }}
                />
              )}
              {buttonState.resume && (
                <PlayCircleOutlined
                  onClick={handleResume}
                  style={{ marginLeft: 10, cursor: "pointer" }}
                />
              )}
              {buttonState.restart && (
                <Popconfirm
                  title="Are you sure you want to Restart?"
                  onConfirm={handleRestart}
                  okText="Yes"
                  cancelText="No"
                >
                  <ReloadOutlined
                    style={{ marginLeft: 10, cursor: "pointer" }}
                  />
                </Popconfirm>
              )}
              {buttonState.cancel && (
                <Popconfirm
                  title="Are you sure you want to cancel?"
                  onConfirm={handleCancel}
                  okText="Yes"
                  cancelText="No"
                >
                  <StopOutlined style={{ marginLeft: 10, cursor: "pointer" }} />
                </Popconfirm>
              )}
              <Progress
                percent={progress}
                style={{ marginTop: 10 }}
                status={
                  reportStatusData?.errorMessage
                    ? "exception"
                    : reportStatusData?.progressPercentage === 100
                    ? "success"
                    : isFetchingStatus
                    ? "active"
                    : "normal"
                }
              />
            </div>
          </div>
        </Card>
      )}

      <EnhancedDrawer
        drawerVisible={drawerVisible}
        drawerMode={drawerMode}
        onClose={onImportCloseClick}
        isLoadingButton={isLoadingButton}
        handleSubmit={handleImportSubmit}
        form={importform}
        commonFields={importCommonFields}
      />
    </>
  );
};

const EnhancedDrawer = withDrawer(Drawer);

export default ReportUpload;
