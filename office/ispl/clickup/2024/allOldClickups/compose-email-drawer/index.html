    Problem

    Scenario

    When searching and selecting the Reference in the compose. The selected reference tags is removed in the selected input when new search term is searched.

    mailMessageReferenceDtos is used in two places useEffect.

    const [mailRefDtos, setmailRefDto] = useState([]);

// 1: linenumber: 257
useEffect(() => {
  const idRef = referenceListingResponse?.items?.find((x) => {
    return x?.code == searchKeywordFormatData;
  });

  form.setFieldsValue({
    // mailMessageReferenceDtos: idRef?.code,
    subject:
      form.getFieldsValue()["subject"] === undefined
        ? searchKeywordFormatData
        : form.getFieldsValue()["subject"],
  });
}, [referenceListingResponse?.items]);

// line number: 2783
useEffect(() => {
  form.setFieldsValue({
    mailMessageReferenceDtos: mailRefDtos,
  });
}, [mailRefDtos]);

// .. return
<Form.Item
